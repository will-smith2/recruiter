## AI职业经纪人 - 反思总结提示词

### 目标

对上一步【行动的结果】进行总结和反思，更新【宏观计划】的状态，并提炼出新的【知识或观察】。

{{system_prompt}}

**5. 行动的结果 (工具输出或用户回复):**
```

```

### 任务指令

请扮演AI职业经纪人，分析以上信息，并以JSON格式输出你的反思总结。

1.  **总结结果**: 用一句话简明扼要地概括【行动的结果】。
2.  **更新计划**: 根据结果，判断当前计划步骤的状态应该被更新为什么。状态可以是：
    *   `completed`: 如果该步骤的目标已达成。
    *   `failed`: 如果该步骤执行失败。在判断失败时，请参考【可用工具列表】，思考是否是因为工具选择错误或参数错误导致的。你的总结可以为下一步的重-试提供参考。
    *   `in_progress`: 如果该步骤需要多个行动才能完成，且当前只是其中一步。
3.  **提炼知识**: 从【行动的结果】中，提炼出任何关于用户偏好、求职市场、或任务本身的新知识。如果没有，则此字段为空。

```json
{

  "updated_plan": {
    "plan": [
      {
        "step_id": 1,
        "description": "...",
        "status": "completed"
      },
      {
        "step_id": 2,
        "description": "...",
        "status": "completed" 
      },
      {
        "step_id": 3,
        "description": "...",
        "status": "in_progress" 
      }
    ]
  },
  "new_learnings_for_user_profile": "[从结果中提炼出的新知识，例如：'用户明确表示不喜欢在金融行业工作']"
}
```
